# fed-geo-2020
Work in Progress: Website for Fed Geo Day 2020 Conference

# install
```bash
bundle install --path vendor/bundle
```

# run
Locally
```
bundle exec jekyll serve
```
If you are on a server, you may have to run:
```
bundle exec jekyll serve --host 0.0.0.0
```
